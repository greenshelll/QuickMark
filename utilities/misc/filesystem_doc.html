Python Code Documentation

FileSystem Class

Represents a file system.

Attributes:

  • sheets: A list of Sheets objects.
  • local_file_directory: The directory where the local file is stored.
  • open_index: Index of the currently open sheet.

Methods:

  • add_sheet: Adds a new sheet to the file system.
  • get_sheet: Retrieves a sheet from the file system.
  • save: Saves the file system object to a pickle file.
  • load: Loads the file system object from a pickle file.

Item Class

Represents an item.

Attributes:

  • points: The points associated with the item.
  • answer_key: The answer key for the item.
  • fs_obj: The file system object associated with the item.

Methods:

  • evaluate: Evaluates the item against the provided answer.

MultipleChoice Class

Represents a multiple-choice question.

Attributes:

  • items: A list of Item objects representing the choices.
  • show_items: The number of choices to display.
  • fs_obj: The file system object associated with the multiple-choice question.

Methods:

  • set_items: Sets the number of choices for the question.
  • get_items: Gets the list of choices for the question.

TrueOrFalse Class

Represents a true or false question.

Attributes:

  • items: A list of Item objects representing the true/false choices.
  • fs_obj: The file system object associated with the true/false question.

Identification Class

Represents an identification question.

Attributes:

  • items: A list of Item objects representing the identification choices.
  • fs_obj: The file system object associated with the identification question.

AnswerKeys Class

Represents the answer keys for different types of questions.

Attributes:

  • mc: The answer key for multiple-choice questions.
  • tf: The answer key for true/false questions.
  • idtf: The answer key for identification questions.
  • fs_obj: The file system object associated with the answer keys.

CheckSession Class

Represents a session for checking answers.

Attributes:

  • date_created: The date and time when the session was created.
  • fs_obj: The file system object associated with the check session.
  • answer_key: The answer key for the session.
  • mc: The multiple-choice questions in the session.
  • tf: The true/false questions in the session.
  • idtf: The identification questions in the session.
  • open_index: Index of the currently open session.

CheckSheets Class

Represents a collection of check sessions.

Attributes:

  • check_sessions: A list of CheckSession objects.
  • fs_obj: The file system object associated with the check sheets.

Sheets Class

Represents a collection of sheets.

Attributes:

  • date_created: The date and time when the sheet was created.
  • name: The name of the sheet.
  • answer_key: The answer key for the sheet.
  • check_sheets: The check sheets associated with the sheet.
  • fs_obj: The file system object associated with the sheet.
